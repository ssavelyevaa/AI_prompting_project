# Prompt Examples

## Task 1: Summarization

### Unstructured Prompt
Summarize this text.

### Observation
- Longer response
- Included extra details
- Less focused

---

### Structured Prompt
Summarize this text in:
1. Main idea  
2. Key points  
3. Why it matters  

### Observation
- Clear sections
- More concise
- Easier to scan
